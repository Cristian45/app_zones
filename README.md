
# App zonas afectadas

This system allows for control and registration of coconut palm areas that are affected by different pests.


## Deployment

To deploy this project run

```bash
  docker-compose up -d
```


## Tech Stack

**Client:** Angular

**Server:** Java, SpringBoot
